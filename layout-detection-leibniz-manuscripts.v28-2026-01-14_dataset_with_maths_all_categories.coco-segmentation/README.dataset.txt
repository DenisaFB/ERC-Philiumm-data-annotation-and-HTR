The ERC Philiumm project aims at transcribing the unpublished mathematical manuscripts of Gottfried Wilhelm Leibniz (1646-1716). We use instance segmentation instead of object detection because of the nature of the manuscripts: the presence of annotations and corrections require a more detailed segmentation. We first trained a Roboflow 3.0 model on 100 labeled images.
